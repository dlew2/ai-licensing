Key Findings for IT Procurement

Google: strongest FERPA story + Workspace bundle value

Google is the only vendor with explicit FERPA support (plus COPPA, FedRAMP High, ISO 27001, SOC 2). If your university already uses Google Workspace, Gemini AI is now bundled into paid plans at no extra cost. The Education plan extends this campus-wide at discounted rates. The AI Studio interface is completely free for prototyping.

Top recommendation for campus-wide deployment

Claude's Education Plan is the only product with an explicit university-wide offering covering students, faculty, and staff at institutional rates. Contact Anthropic sales directly — this plan is not available through individual signups.

Compliance strength: Claude wins clearly

Claude Enterprise offers the deepest compliance stack: SOC 2 Type II, ISO 27001, HIPAA-ready with BAA, Zero Data Retention (ZDR), SCIM, GDPR DPA, and IP allowlisting. Cursor offers SOC 2 and SSO but lacks HIPAA and ZDR. None offer explicit FERPA certification.

FERPA: negotiate directly

No vendor offers FERPA certification as a checkbox. Universities must negotiate a Data Processing Agreement (DPA) with each vendor and contractually exclude student data from model training. Claude's ZDR addendum is the strongest tool available for this.

Claude Code is not separate

Claude Code is included in Claude Pro and above — not a standalone purchase. For a development team, Team Premium seats ($100/seat/mo annual) add Claude Code. Non-developer staff can use cheaper Standard seats ($25/seat/mo annual) in the same plan.

Cursor: best for developer-only teams

Cursor Teams ($40/user/mo) is compelling for CS/IT engineering departments needing a dedicated AI code editor with SSO and admin controls. It is not appropriate for institution-wide deployment due to limited compliance certifications.

Cost predictability: watch Cursor credits

Cursor switched to a credit-based model in mid-2025. Costs are variable — developers using premium models like Claude Sonnet manually can exhaust their monthly credit pool quickly. Cursor's Auto mode is unlimited but uses lower-cost models. Budget accordingly.
